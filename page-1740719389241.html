Frequently Asked Questions

To add a new product, follow these steps:

  1. Click on the "Add Product" button in the top right of the products list
  2. Fill in the product name and description
  3. Upload an image or video for the product (optional)
  4. Add relevant tags to help categorize the product
  5. Click "Add Product" to save

To edit or delete a product:

  • Edit: Click the edit (pencil) icon on any product card, or click on a product to view it and then select "Edit Product". Make your changes and click "Update Product".
  • Delete: To delete a product, either:
    • Click the delete (trash) icon on the product card
    • Open the edit product form and click "Delete Product"
    • View the product and click the delete button
    A confirmation dialog will appear before deleting.

The following file types are supported:

Images:

JPG, JPEG, PNG, GIF, WebP

Maximum file size: 10MB (can be adjusted in settings)

Videos:

MP4, WebM, Ogg

Maximum file size: 50MB (can be adjusted in settings)

Tags help you categorize and filter products. Here's how to use them effectively:

  • Adding tags: When creating or editing a product, type a tag name in the tags field and press Enter, or click the + button.
  • Removing tags: Click the X icon next to any tag to remove it.
  • Default tags: You can set up common tags in the Settings page that will be available for quick selection.
  • Filtering by tags: On the products list page, use the tag filter dropdown to show only products with specific tags.

Yes, you can import and export product data:

Import:

Go to Settings → Data Management → Import. You can upload a JSON or CSV file with product data.

Export:

Go to Settings → Data Management → Export. You can download your product data as JSON or CSV format.

Note: Media files (images and videos) need to be imported separately. The export will include URLs to the media files.
